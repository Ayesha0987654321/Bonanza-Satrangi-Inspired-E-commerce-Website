## 📖 About the Project

This project is a front-end inspired replica of the Bonanza Satrangi e-commerce platform, built to practice and demonstrate core web development skills. It covers semantic HTML structure, responsive CSS layouts, and interactive JavaScript features — delivering a realistic shopping experience from browsing collections to exploring individual product pages.

---

## ✨ Features

- 🏠 Home Page — Hero banner, featured collections, and promotional sections
- 👗 Product Listings — Browse clothing by category with clean card layouts
- 🔍 Product Detail Pages** — Individual product views with images and descriptions
- 🛒 Shopping Cart UI — Interactive cart interface for adding and removing items
- 📱 Responsive Design — Fully optimized for mobile, tablet, and desktop screens
- 🎨 Vibrant UI — Color palette and layout inspired by Bonanza Satrangi's brand identity
- 🔗 Multi-page Navigation — Smooth navigation across multiple HTML pages

## 🚀 Getting Started

No installation or build tools required. Follow these simple steps to run the project locally:

### Prerequisites

- A modern web browser (Chrome, Firefox, Edge, or Safari)
- [VS Code](https://code.visualstudio.com/) with the **Live Server** extension *(recommended)*

### Steps

1. Clone the repository:

```bash
git clone https://github.com/Ayesha0987654321/Bonanza-Satrangi-Inspired-E-commerce-Website.git
```

2. Navigate into the project folder:

```bash
cd Bonanza-Satrangi-Inspired-E-commerce-Website
```

3. Open the project:

Simply open any `.html` file from the `html/` folder in your browser — or for a better experience, use the Live Server extension in VS Code:

```
Right-click on any .html file → Open with Live Server
```

No dependencies to install. That's it! 🎉
